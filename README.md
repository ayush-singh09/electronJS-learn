## You can use this repo as the base for your electron projects


**✨ Scripts -**

dev - running full application with one command (vite running at 5173)

dev:react - running only vite

dev:electron - running only   electron

transpile:electron - Transpiling electron's ts files into js file

dist:mac, dist:win, dist:linux - Build the whole code and give the final application (different for each OS)
